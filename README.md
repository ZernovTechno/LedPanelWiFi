# Упрощенная установка:

1. Установить драйвер CH340, используя файл CH340.exe
2. Запустить FlashDownloadTool.exe
3. Выбрать ESP8266
4. Залить LedPanelWiFi.bin по адресу 0x00
5. Загрузить Data, используя инструкцию
