Здесь лежат дополнительные файлы, которые могут быть использованы в некоторых конфигурациях:

Папка "firmware/LedPanelWiFi_v1.14/data"

  Содержит дополнительные файлы, которые могут быть полезны для матриц 16x16 и большего размера
  Нужные файлы и папки нужно скопировать в папку проекта "firmware/LedPanelWiFi_v1.14/data"
  Они будут загружены в файловую систему микроконтроллера при выполнении операции загрузки данных в микроконтроллер
  https://github.com/vvip-68/LedPanelWiFi/wiki/Загрузка-данных-в-файловую-систему-МК

    Папка 16p16 - содержит файлы картинок для эффекта "Слайды"; 

    Папка 16x16 - содержит файлы роликов из Jinx, которыt могут быть использованы на матрицах 16x16 без SD-карты
                  при включенной в настройках скетча опции эмуляции SD-карты в файловой системе:
                     #define USE_SD 0    // поставьте 0, если у вас нет SD-card модуля
                     #define FS_AS_SD 1  // при USE_SD == 1; FS_AS_SD == 0 - эффекты на SD-карте, 1 - эффекты в файловой ситеме МК, 
                                         // самого SD-card модуля нет (режим эмуляции SD-карты для эффектов 'SD-карта')

                     В папке только несколько роликов для примера, поскольку общий размер файловой системы ограничен 2MB на Wemos / Node MCU
                     и 1.2 МБ на ESP32. 

                     В эти 2 Mb должна вмещаться папка Web (обязательная), иметь около 120 кб для бэкапа.
                     Остальное место может быть использовано по желанию пользователя, например для коротких роликов из Jinx!

                     Дополнительные ролики - по ссылке: https://drive.google.com/drive/folders/1NCXAp-3A5JxJseGmKwkXySUwA-kVNXEN?usp=sharing

                     Если при попытке загрузить файловую систему в контроллер загрузчик пишет ошибку - недостаточно места - 
                     удаляйте файлы из этой папки, пока загрузка не будет выполнена успешно


    Файлы карт индексов  https://github.com/vvip-68/LedPanelWiFi/wiki/Сборная-матрица,-состоящая-из-матриц-разного-типа-и-размера

    Файл 16x16.map - пример карты индексов для матрицы 16x16. Может быть просмотрена утилитой LedMapper из папки проекта Tools
                     Карта индексов используется для нестандартных и/или матриц, когда невозможно использовать 
                     описание марицы через угол подключения и направдение из угла.

                     Задается в настройках скетча:
                     
                     #define MATRIX_TYPE 2           // тип матрицы: 0 - зигзаг, 1 - параллельная, 2 - использовать карту индексов
                     #define MATRIX_INDEX 1          // только для MATRIX_TYPE == 2 - место расположения массива индексов 0 - в сектче, 1 - в файле  

                     Файл долден быть размещен в корне файловой системе микроконтроллера.
                     Для загрузки в файловую систему скопируйте файл в папку "firmware/LedPanelWiFi_v1.14/data"
                  
    Файл 32x16.map - пример карты индексов для матрицы 32x16. Назначение и способ использования - тот же, сто для файла 16x16.map - см. описание выше

    Файл 48x16.map - пример карты индексов для матрицы 48x16. Назначение и способ использования - тот же, сто для файла 16x16.map - см. описание выше
